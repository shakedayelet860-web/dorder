# 甜蜜点餐 - 5分钟快速部署教程

## 🎯 目标
让你和女朋友两个人能真正用起来，互相点菜！

---

## 方案一：最简单（推荐）- 使用 Vercel 免费部署

### 步骤1：部署前端（2分钟）

1. **注册 Vercel 账号**
   - 访问 https://vercel.com
   - 用 GitHub 账号登录

2. **导入项目**
   - 点击 "Add New Project"
   - 导入你的代码仓库
   - 框架选择 "Vite"
   - 点击 "Deploy"

3. **获得网址**
   - 部署完成后会得到类似 `https://couple-order-xxx.vercel.app` 的地址
   - 这个地址就是你们的APP入口！

### 步骤2：部署后端（3分钟）

1. **在 Vercel 创建另一个项目**
   - 进入 `server` 文件夹
   - 同样导入到 Vercel
   - 获得后端地址如 `https://couple-order-server-xxx.vercel.app`

2. **更新前端配置**
   - 修改 `src/config.ts` 中的服务器地址
   - 重新部署前端

### 步骤3：开始使用！

1. **创建房间**
   - 你打开APP，点击"创建房间"
   - 获得一个6位房间号（如：ABC123）

2. **女朋友加入**
   - 把房间号发给女朋友
   - 她打开同一个网址，输入房间号加入

3. **开始点菜**
   - 你们可以实时看到对方的订单
   - 订单状态实时同步

## 方案二：本地使用（无需部署）

如果你们在同一个WiFi下，可以直接使用：

### 步骤1：启动后端
```bash
cd server
npm install
npm start
```

### 步骤2：启动前端
```bash
npm run dev -- --host
```

### 步骤3：手机访问
- 查看终端显示的 `Network` 地址
- 如：`http://192.168.1.100:3000/`
- 你和女朋友都用手机浏览器访问这个地址
- 创建/加入房间即可使用

**缺点**：必须同一WiFi，不能远程使用

## 方案三：使用 Render/Railway 部署（免费）

如果 Vercel 部署后端有问题，可以用这些平台：

### Render（推荐）
1. 访问 https://render.com
2. 注册账号
3. 创建 Web Service
4. 选择你的代码仓库
5. 根目录填写 `server`
6. 构建命令：`npm install`
7. 启动命令：`npm start`
8. 免费部署完成！

### Railway
1. 访问 https://railway.app
2. 类似步骤部署

## 📱 手机安装为APP

### Android Chrome：
1. 打开部署好的网址
2. 点击右上角菜单（三个点）
3. 选择"添加到主屏幕"
4. 完成！像普通APP一样使用

### iOS Safari：
1. 打开部署好的网址
2. 点击底部分享按钮
3. 选择"添加到主屏幕"
4. 完成！

## 🔧 关键配置

### 1. 修改服务器地址

创建 `src/config.ts`：
```typescript
// 本地开发
export const SERVER_URL = 'http://localhost:3001';

// 部署后改成你的服务器地址
// export const SERVER_URL = 'https://your-server.vercel.app';
```

### 2. 环境变量（可选）

创建 `.env` 文件：
```
VITE_SERVER_URL=https://your-server.vercel.app
```

## 💡 常见问题

### Q: 部署后两个人数据不同步？
A: 检查服务器地址配置是否正确，确保两个人访问的是同一个后端。

### Q: 手机收不到通知？
A: 需要在手机设置中允许浏览器发送通知。

### Q: 离线能用吗？
A: 可以查看菜单和教程，但订单同步需要网络。

### Q: 数据会丢失吗？
A: 服务器重启后数据会清空。如需持久化，需要添加数据库（如MongoDB）。
